Script API

Your Python script must have a function called nextStep(). This function should return one of the following predefined values

  • Forward
    • Moves you one cell forward.
  • Backward
    • Moves you one cell backward.
  • Left
    • Rotates you 90 degrees to the left.
  • Right
    • Rotates you 90 degrees to the right.

You are given a predefined variable called maze which provides methods to observe the world around you.
